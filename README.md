# Decntralized Clous Storage using IPFS and Blockchain

Build and deploy : Decntralized Clous Storage using IPFS and Blockchain.

Develop a decentralized platform for managing and verifying digital asset licenses.
Integrate IPFS via Pinata for secure and distributed file storage.

Get your own Pinata API Keys and SECRET API Key, and replace them in the function that uploads the files and user data.

```https://www.pinata.cloud/
  API Key: YOUR KEY
  SECRET API Key: YOUR KEY
```
